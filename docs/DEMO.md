# VoiceTyping 功能演示稿

> 建议演示时长：8–12 分钟  
> 演示前准备：启动程序、打开记事本、确认麦克风可用

---

## 开场（30 秒）

大家好，这是我做的 **VoiceTyping 语音输入法**。

它解决的核心问题是：**在任意 Windows 应用里，用语音快速输入文字**。

和云端方案不同，VoiceTyping 使用本地 Whisper 模型识别语音，**不需要 API 密钥，首次下载模型后可以离线使用，数据也不上传云端**。

---

## 一、启动程序（30 秒）

**操作：**
1. 运行 `VoiceTyping.exe` 或 `python -m voicetyping.main`
2. 指向系统托盘图标

**讲解词：**
> 启动后程序在后台运行，托盘图标显示当前状态。  
> 灰色代表就绪，红色代表正在录音，蓝色代表正在识别。  
> 首次运行需要联网下载约 145 MB 的语音识别模型，之后就可以离线使用了。

---

## 二、基础语音输入（1 分钟）★ 核心功能

**操作：**
1. 打开**记事本**，点击输入区域
2. **按住** `Ctrl + Shift + Space`
3. 说：「今天天气很好，我正在演示 VoiceTyping 语音输入法。」
4. **松开**快捷键

**讲解词：**
> 这是 Push-to-Talk 模式：按住快捷键开始说话，松开后自动识别并插入文字。  
> 文字会通过剪贴板粘贴到当前光标位置，所以在 Word、浏览器、微信里都可以使用。

**预期效果：** 文字出现在记事本中。

---

## 三、录音提示与音量检测（1 分钟）

**操作：**
1. 再次按住热键，指向屏幕底部悬浮条
2. 故意离麦克风远一点、小声说话后松开

**讲解词：**
> 录音时屏幕底部会显示「麦克风使用中」，并有实时音量条，让用户知道麦克风正在工作。  
> 如果声音太小，程序会弹出警告和系统通知，提醒用户靠近麦克风或调高输入音量。

---

## 四、历史剪贴板（1 分钟）

**操作：**
1. **左键点击**托盘图标，打开历史剪贴板
2. 展示刚才的识别记录
3. 在搜索框输入「天气」
4. 选中一条记录，点击「复制」或「插入到光标」

**讲解词：**
> 每次语音转换的结果都会自动保存到历史剪贴板。  
> 支持关键词搜索，也可以把历史内容重新复制或插入到当前应用中。  
> 窗口打开时，新的识别记录会实时刷新。

---

## 五、简繁体切换（30 秒）

**操作：**
1. 右键托盘 → **文字输出** → 选择 **繁體中文**
2. 按住热键说：「软件工程是一门有趣的学科。」
3. 松开后展示繁体输出

**讲解词：**
> 识别完成后可以自动转换为简体或繁体中文，在托盘菜单里一键切换，适合不同书写习惯的用户。

**可选：** 再切回「简体中文」演示一次。

---

## 六、自定义热键（30 秒）

**操作：**
1. 右键托盘 → **修改热键**
2. 点击「录制新快捷键」
3. 按下例如 `Ctrl + Alt + V`，保存

**讲解词：**
> 用户可以根据自己的习惯自定义快捷键，设置后立即生效，并保存到配置文件中。

---

## 七、麦克风选择（30 秒）

**操作：**
1. 右键托盘 → **选择麦克风**
2. 展示设备列表，选择不同设备并保存

**讲解词：**
> 笔记本通常有多个麦克风设备，用户可以在设置里切换使用内置麦克风或外接耳机麦克风。

---

## 八、本地识别与离线能力（30 秒）

**操作：** 无需操作，口头说明即可。

**讲解词：**
> VoiceTyping 基于 faster-whisper 在本地完成语音识别。  
> 首次运行下载模型后，日常使用不需要联网，也不产生 API 费用。  
> 配置和识别历史都保存在本机 `%APPDATA%\VoiceTyping\` 目录下。

---

## 九、打包分发（30 秒，可选）

**操作：** 展示 `dist\VoiceTyping\` 文件夹或安装包。

**讲解词：**
> 项目支持 PyInstaller 打包成 exe，也可以通过 Inno Setup 制作安装包。  
> 普通用户无需安装 Python，解压或安装后双击即可使用。

---

## 结束语（20 秒）

**讲解词：**
> 总结一下，VoiceTyping 的核心优势是：**本地识别、隐私安全、零 API 成本、全局快捷键、任意应用输入**。  
> 适合需要大量文字输入、又希望数据留在本地的 Windows 用户。  
> 谢谢大家，欢迎提问。

---

## 演示检查清单

演示前请确认：

- [ ] 程序已启动，托盘图标可见
- [ ] 模型已下载完成（首次演示需提前联网下载）
- [ ] 记事本或其他文本编辑器已打开
- [ ] 麦克风权限正常
- [ ] 建议以**管理员身份**运行（避免热键失效）
- [ ] 系统音量 / 输入设备音量正常

---

## 常见问题应急

| 情况 | 应对 |
|------|------|
| 热键无反应 | 说明需管理员权限，改用管理员重新启动 |
| 识别结果为空 | 靠近麦克风，说长一点（>0.3 秒） |
| 首次启动慢 | 提前说明正在加载本地模型 |
| 网络问题 | 说明仅首次下载模型需要联网 |

---

## 精简版（5 分钟）

如果时间有限，只演示以下 4 步：

1. **启动** → 展示托盘图标  
2. **语音输入** → 记事本中按住热键说话（核心）  
3. **历史剪贴板** → 左键打开，搜索历史记录  
4. **简繁切换** → 托盘菜单切换并再输入一句  

结束语：本地识别、离线可用、零成本、全局输入。
